Γλωσσάρι Web Development

Στον παρακάτω πίνακα θα βρείτε όλους τους βασικούς και προχωρημένους όρους που χρησιμοποιούνται στην ανάπτυξη ιστοσελίδων, με αγγλική και ελληνική ονομασία και σύντομη επεξήγηση.
* Όροι με αστερίσκο: Εξειδικευμένοι ή προχωρημένοι όροι.

HTML
Αγγλικός Όρος Ελληνικός Όρος Περιγραφή
HTMLΥπερκείμενοΓλώσσα σήμανσης για τη δομή και το περιεχόμενο ιστοσελίδων.
TagΕτικέταΣτοιχείο HTML που ορίζει τμήμα της δομής (π.χ. <h1>, <p>).
ElementΣτοιχείοΠλήρης μονάδα HTML (π.χ. <div>Κείμενο</div>).
AttributeΙδιότηταΠρόσθετη πληροφορία σε ετικέτα (π.χ. src, alt, href).
HeadΚεφαλίδαΤμήμα HTML με μεταδεδομένα, τίτλο, συνδέσεις σε CSS/JS.
BodyΣώμαΤο κύριο ορατό περιεχόμενο της ιστοσελίδας.
DoctypeΔήλωση τύπουΕνημερώνει τον browser για την έκδοση HTML.
Semantic Tags *Σημασιολογικές ετικέτεςΕτικέτες που περιγράφουν το ρόλο του περιεχομένου (π.χ. <header>, <nav>, <main>, <footer>).
FormΦόρμαΕνότητα για συλλογή δεδομένων από τον χρήστη.
InputΠεδίο εισαγωγήςΣτοιχείο φόρμας για λήψη δεδομένων από τον χρήστη.
AnchorΣύνδεσμοςΕτικέτα για υπερσυνδέσμους (<a href="...">).
ImageΕικόναΕνσωμάτωση εικόνας στη σελίδα (<img src="...">).
ListΛίσταΟμαδοποίηση στοιχείων (π.χ. <ul>, <ol>).
TableΠίνακαςΔομή για εμφάνιση δεδομένων σε γραμμές και στήλες.
CommentΣχόλιοΜη ορατό κείμενο για σημειώσεις στον κώδικα (<!-- σχόλιο -->).
MetaΜεταδεδομέναΠληροφορίες για τη σελίδα (π.χ. charset, viewport).
Alt AttributeΕναλλακτικό κείμενοΚείμενο που εμφανίζεται αν δεν φορτωθεί η εικόνα, χρήσιμο για προσβασιμότητα.
Section *ΕνότηταΧωρίζει το περιεχόμενο σε ενότητες.
Article *ΆρθροΑυτόνομο κομμάτι περιεχομένου.
Aside *Πλαϊνή ενότηταΠληροφορίες στο πλάι (π.χ. sidebar).
Header *ΚεφαλίδαΗ κεφαλίδα της σελίδας ή μιας ενότητας.
Footer *ΥποσέλιδοΤο υποσέλιδο της σελίδας.
Nav *ΠλοήγησηΤο μενού πλοήγησης.
Viewport *Περιοχή προβολήςΤο ορατό τμήμα της σελίδας στη συσκευή.
CSS
Αγγλικός Όρος Ελληνικός Όρος Περιγραφή
CSSΦύλλα στυλ (CSS)Γλώσσα για τη μορφοποίηση και το στυλ των ιστοσελίδων.
SelectorΕπιλογέαςΕπιλέγει ποια στοιχεία HTML θα μορφοποιηθούν.
PropertyΙδιότηταΧαρακτηριστικό που ορίζει το στυλ (π.χ. color, font-size).
ValueΤιμήΗ τιμή μιας ιδιότητας CSS (π.χ. color: red;).
ClassΚλάσηΟμάδα στυλ που εφαρμόζεται με το class="...".
IDΤαυτότηταΜοναδικό αναγνωριστικό για ένα στοιχείο.
Box Model *Μοντέλο ΚουτιούΤρόπος υπολογισμού διαστάσεων στοιχείου (content, padding, border, margin).
MarginΕξωτερικό περιθώριοΑπόσταση έξω από το στοιχείο.
PaddingΕσωτερικό περιθώριοΑπόσταση μέσα από το στοιχείο, γύρω από το περιεχόμενο.
BorderΠεριγράμματαΓραμμή γύρω από το στοιχείο.
ResponsiveΠροσαρμοστικότηταΙκανότητα του layout να αλλάζει ανάλογα με τη συσκευή.
Media Query *Ερώτημα πολυμέσωνΚανόνας CSS που εφαρμόζεται σε συγκεκριμένα μεγέθη οθόνης.
Flexbox *Ευέλικτο πλαίσιοΣύστημα διάταξης για εύκολη στοίχιση και ευελιξία στοιχείων.
Grid *ΠλέγμαΣύστημα διάταξης για δισδιάστατη τοποθέτηση στοιχείων.
HoverΕφέ κατάδειξηςΕφέ που εφαρμόζεται όταν το ποντίκι περνάει πάνω από ένα στοιχείο.
Responsive ImageΠροσαρμοζόμενη εικόναΕικόνα που αλλάζει μέγεθος ανάλογα με το πλάτος του κοντέινερ.
Gap *ΚενόΑπόσταση μεταξύ flex items ή grid cells.
Breakpoint *Σημείο αλλαγήςΣυγκεκριμένο πλάτος οθόνης όπου αλλάζει το layout.
Mobile-first *Πρώτα για κινητάΣχεδιασμός που ξεκινά από μικρές οθόνες και επεκτείνεται.
JavaScript
Αγγλικός Όρος Ελληνικός Όρος Περιγραφή
JavaScriptΤζάβασκριπτΓλώσσα προγραμματισμού για διαδραστικότητα στις ιστοσελίδες.
VariableΜεταβλητήΑποθηκεύει τιμές (π.χ. let x = 5;).
FunctionΣυνάρτησηΟμάδα εντολών που εκτελούν μια εργασία.
EventΣυμβάνΕνέργεια που συμβαίνει στη σελίδα (π.χ. click, submit).
DOM *Μοντέλο Αντικειμένου ΕγγράφουΗ δομή της HTML όπως την "βλέπει" η JavaScript.
ArrayΠίνακαςΔομή που αποθηκεύει πολλαπλές τιμές.
ObjectΑντικείμενοΣυλλογή ιδιοτήτων και τιμών.
LoopΒρόχοςΕπαναλαμβάνει εντολές (π.χ. for, while).
ConditionΣυνθήκηΈλεγχος λογικής (π.χ. if, else).
Callback *Συνάρτηση επανάκλησηςΣυνάρτηση που εκτελείται μετά από άλλη συνάρτηση.
Event Listener *Ακροατής συμβάντοςΚώδικας που "ακούει" για συμβάντα (π.χ. addEventListener).
AJAX *Ασύγχρονη ανταλλαγή δεδομένωνΤεχνική για επικοινωνία με server χωρίς ανανέωση σελίδας.
Debugging *Εντοπισμός σφαλμάτωνΔιαδικασία εύρεσης και διόρθωσης λαθών στον κώδικα.
ConsoleΚονσόλαΕργαλείο για εμφάνιση μηνυμάτων και debugging.
PHP
Αγγλικός Όρος Ελληνικός Όρος Περιγραφή
PHPΠι-Ειτς-Πι (PHP)Γλώσσα προγραμματισμού για δυναμικές σελίδες στον server.
Server-sideΠλευρά διακομιστήΕκτέλεση κώδικα στον server πριν σταλεί στον browser.
VariableΜεταβλητήΑποθήκευση τιμών ($name = "Γιάννης";).
EchoΕκτύπωσηΕντολή για εμφάνιση αποτελέσματος (echo "Γειά σου";).
Include/Require *Ενσωμάτωση αρχείουΕισαγωγή άλλου αρχείου PHP στον τρέχοντα κώδικα.
Form HandlingΕπεξεργασία φόρμαςΔιαχείριση δεδομένων που στέλνει ο χρήστης μέσω HTML φόρμας.
Prepared Statement *Προετοιμασμένη δήλωσηΤεχνική για ασφαλή εκτέλεση SQL εντολών.
CRUD *CRUD (Create, Read, Update, Delete)Βασικές λειτουργίες διαχείρισης δεδομένων σε βάσεις.
MySQL / SQL
Αγγλικός Όρος Ελληνικός Όρος Περιγραφή
MySQLΜάι-Ες-Κιου-ΕλΣύστημα διαχείρισης σχεσιακών βάσεων δεδομένων.
SQLΔομημένη Γλώσσα ΕρωτημάτωνΓλώσσα για διαχείριση και ανάκτηση δεδομένων από βάσεις.
TableΠίνακαςΟργανωμένη αποθήκευση δεδομένων σε γραμμές και στήλες.
RowΓραμμήΜία εγγραφή σε πίνακα.
ColumnΣτήληΈνα πεδίο δεδομένων σε πίνακα.
QueryΕρώτημαΕντολή SQL για λήψη ή τροποποίηση δεδομένων.
SELECTΕπιλογήΕντολή SQL για ανάκτηση δεδομένων.
INSERTΕισαγωγήΕντολή SQL για προσθήκη νέων δεδομένων.
UPDATEΕνημέρωσηΕντολή SQL για αλλαγή υπαρχόντων δεδομένων.
DELETEΔιαγραφήΕντολή SQL για διαγραφή δεδομένων.
Primary KeyΠρωτεύον ΚλειδίΜοναδικό αναγνωριστικό κάθε γραμμής σε πίνακα.
Foreign Key *Ξένο ΚλειδίΣύνδεση μεταξύ πινάκων.
DatabaseΒάση δεδομένωνΣυλλογή οργανωμένων πινάκων και δεδομένων.
ConnectionΣύνδεσηΗ διαδικασία σύνδεσης της PHP με τη βάση δεδομένων.
Responsive Design
Αγγλικός Όρος Ελληνικός Όρος Περιγραφή
Responsive DesignΑνταποκρινόμενος ΣχεδιασμόςΠροσαρμογή της σελίδας σε διαφορετικά μεγέθη οθόνης.
Media Query *Ερώτημα πολυμέσωνCSS κανόνας για διαφορετικά μεγέθη οθόνης.
Mobile-first *Πρώτα για κινητάΣχεδιασμός που ξεκινά από μικρές οθόνες και επεκτείνεται.
Breakpoint *Σημείο αλλαγήςΣυγκεκριμένο πλάτος οθόνης όπου αλλάζει το layout.
Hamburger Menu *Μενού HamburgerΕικονίδιο με τρεις γραμμές για μενού σε κινητά.
Viewport *Περιοχή προβολήςΤο ορατό τμήμα της σελίδας στη συσκευή.
Responsive ImageΠροσαρμοζόμενη εικόναΕικόνα που αλλάζει μέγεθος ανάλογα με το πλάτος του κοντέινερ.
Web Design
Αγγλικός Όρος Ελληνικός Όρος Περιγραφή
Web DesignΣχεδιασμός ΙστοσελίδωνΗ διαδικασία δημιουργίας της εμφάνισης και λειτουργίας ιστοσελίδων.
UIΔιεπαφή ΧρήστηΗ οπτική εμφάνιση και τα διαδραστικά στοιχεία μιας εφαρμογής.
UXΕμπειρία ΧρήστηΗ συνολική εμπειρία και αίσθηση του χρήστη κατά την πλοήγηση.
Accessibility *ΠροσβασιμότηταΗ δυνατότητα χρήσης της σελίδας από άτομα με ειδικές ανάγκες.
SEO *Βελτιστοποίηση για Μηχανές ΑναζήτησηςΤεχνικές για καλύτερη κατάταξη στις μηχανές αναζήτησης.
ARIA *ARIA (Accessible Rich Internet Applications)Ετικέτες και ιδιότητες για βελτίωση της προσβασιμότητας.
Flexbox
Αγγλικός Όρος Ελληνικός Όρος Περιγραφή
FlexboxΕυέλικτο ΠλαίσιοCSS σύστημα διάταξης για ευέλικτη στοίχιση και διανομή χώρου.
Flex ContainerΚοντέινερ FlexΤο στοιχείο που περιέχει τα flex items (display: flex).
Flex ItemΣτοιχείο FlexΤα παιδιά του flex container.
Justify ContentΣτοίχιση Κύριου ΆξοναΕυθυγράμμιση στοιχείων οριζόντια.
Align ItemsΣτοίχιση Εγκάρσιου ΆξοναΕυθυγράμμιση στοιχείων κάθετα.
Flex DirectionΚατεύθυνση FlexΟρίζει αν τα στοιχεία μπαίνουν σε σειρά ή στήλη.
Flex WrapΑναδίπλωση FlexΤα στοιχεία "σπάνε" σε νέες γραμμές αν δεν χωράνε.
GapΚενόΑπόσταση μεταξύ flex items.
Εργαλεία & Περιβάλλον Ανάπτυξης
Αγγλικός Όρος Ελληνικός Όρος Περιγραφή
IDE *Περιβάλλον ανάπτυξηςΕφαρμογή για συγγραφή, δοκιμή και debugging κώδικα.
WebStorm *WebStormΕπαγγελματικό IDE για ανάπτυξη web εφαρμογών.
Live Server *Live ServerΕργαλείο για τοπική προεπισκόπηση και αυτόματη ανανέωση σελίδας.
XAMPP *XAMPPΠακέτο με Apache, MySQL, PHP για τοπικό server.
MAMP *MAMPΠακέτο με Apache, MySQL, PHP για Mac.
Laragon *LaragonΕναλλακτικό πακέτο για τοπικό server σε Windows.
Apache HTTP Server *Apache HTTP ServerΛογισμικό διακομιστή για τοπικό ή online hosting.
BrowserΦυλλομετρητήςΕφαρμογή για προβολή ιστοσελίδων (Chrome, Firefox κ.ά.).
File StructureΔομή αρχείωνΗ οργάνωση των φακέλων και αρχείων στο project.
Pixabay *PixabayΔωρεάν πλατφόρμα εικόνων για χρήση σε ιστοσελίδες.
WordPress WordPress Δημοφιλές σύστημα διαχείρισης περιεχομένου (CMS) για εύκολη δημιουργία και διαχείριση ιστοσελίδων.
CMS (Content Management System) Σύστημα Διαχείρισης Περιεχομένου Λογισμικό για εύκολη δημιουργία, επεξεργασία και διαχείριση ιστοσελίδων.
XAMPP XAMPP Πακέτο τοπικού server με Apache, MySQL/MariaDB, PHP και phpMyAdmin για ανάπτυξη και δοκιμές στον υπολογιστή.
Apache Apache Διακομιστής web που εξυπηρετεί ιστοσελίδες τοπικά ή online.
MySQL/MariaDB MySQL/MariaDB Σύστημα διαχείρισης βάσεων δεδομένων που χρησιμοποιεί το WordPress.
phpMyAdmin phpMyAdmin Εργαλείο για διαχείριση βάσεων δεδομένων μέσω browser.
localhost localhost Η διεύθυνση του τοπικού υπολογιστή (127.0.0.1), για δοκιμές χωρίς σύνδεση στο διαδίκτυο.
Database Βάση δεδομένων Αποθηκεύει όλα τα δεδομένα του WordPress (άρθρα, σελίδες, χρήστες, ρυθμίσεις).
wp-admin wp-admin Η διαχειριστική περιοχή του WordPress (προσβάσιμη μέσω /wp-admin).
wp-content wp-content Φάκελος του WordPress όπου αποθηκεύονται θέματα, πρόσθετα και αρχεία πολυμέσων.
Theme Θέμα Ορίζει την εμφάνιση και τη διάταξη του WordPress site.
Plugin Πρόσθετο Επεκτείνει τη λειτουργικότητα του WordPress (φόρμες, SEO, ασφάλεια, κ.λπ.).
Widget Widget Μικρά εργαλεία που προσθέτουν λειτουργίες σε περιοχές του site (π.χ. sidebar, footer).
Permalink Μόνιμος σύνδεσμος Η σταθερή διεύθυνση URL κάθε σελίδας ή άρθρου στο WordPress.
Dashboard Πίνακας ελέγχου Το κεντρικό σημείο διαχείρισης του WordPress.
Post Άρθρο Δημοσίευση blog ή είδησης στο WordPress.
Page Σελίδα Στατική σελίδα (π.χ. Σχετικά, Επικοινωνία) στο WordPress.
Menu Μενού Πλοήγηση της ιστοσελίδας με συνδέσμους προς σελίδες/άρθρα.
Customizer Προσαρμογέας Εργαλείο του WordPress για παραμετροποίηση εμφάνισης σε πραγματικό χρόνο.
Elementor Elementor Δημοφιλές πρόσθετο για δημιουργία σελίδων με drag & drop.
Shortcode Shortcode Μικρός κώδικας σε αγκύλες που προσθέτει λειτουργίες ή περιεχόμενο σε σελίδες/άρθρα.
Media Library Βιβλιοθήκη πολυμέσων Αποθήκευση και διαχείριση εικόνων, βίντεο, αρχείων στο WordPress.
User Role Ρόλος χρήστη Καθορίζει τα δικαιώματα κάθε χρήστη (Διαχειριστής, Συντάκτης, κ.λπ.).
Backup Αντίγραφο ασφαλείας Αποθήκευση όλων των δεδομένων του site για επαναφορά σε περίπτωση προβλήματος.
SSL Πιστοποιητικό SSL Εξασφαλίζει ασφαλή σύνδεση (https) στο site.
SEO Βελτιστοποίηση για Μηχανές Αναζήτησης Τεχνικές για καλύτερη κατάταξη στις μηχανές αναζήτησης.
Staging Site Πειραματικό site Αντίγραφο του site για δοκιμές πριν τις αλλαγές στο κανονικό site.
FTP FTP (Πρωτόκολλο μεταφοράς αρχείων) Χρήσιμο για μεταφορά αρχείων από/προς τον server.
Child Theme Θέμα παιδί Θέμα που βασίζεται σε άλλο (γονικό) θέμα για ασφαλείς παραμετροποιήσεις.

Responsive Ιστοσελίδες - Σχεδιασμός για Κινητές Συσκευές

Σε αυτή την ενότητα θα μάθετε πώς να σχεδιάζετε responsive ιστοσελίδες (ανταποκρινόμενες ιστοσελίδες), ώστε να προσαρμόζονται ομαλά σε διάφορες συσκευές, όπως κινητά τηλέφωνα, tablets και υπολογιστές.

Τι είναι το Responsive Design;

Το responsive web design είναι η τεχνική που επιτρέπει σε μια ιστοσελίδα να αλλάζει εμφάνιση και διάταξη ανάλογα με το μέγεθος της οθόνης του χρήστη. Έτσι, όλοι οι επισκέπτες έχουν ευχάριστη εμπειρία, είτε χρησιμοποιούν κινητό, tablet ή υπολογιστή.

  • Βελτιώνει την προσβασιμότητα και την εμπειρία χρήστη.
  • Είναι απαραίτητο για SEO (καλή κατάταξη στη Google).
  • Αποφεύγει την ανάγκη για ξεχωριστές mobile και desktop εκδόσεις.

Χρήση Ετικέτας Viewport και CSS Media Queries

<!DOCTYPE html>
<html lang="el">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="style.css">
    <title>Responsive Σελίδα</title>
  </head>
  <body>
    <h1>Καλώς ήρθατε!</h1>
  </body>
</html>

Η ετικέτα <meta name="viewport"> είναι απαραίτητη για την υποστήριξη responsive σχεδίασης. Ορίζει πώς προβάλλεται η ιστοσελίδα σε διαφορετικές διαστάσεις οθονών.

Με media queries στο CSS μπορείτε να προσαρμόσετε το layout: 

/* Στυλ για οθόνες με πλάτος έως 768px */
@media (max-width: 768px) {
  body {
    font-size: 16px;
    background-color: #f0f0f0;
  }

  nav .nav-links {
    display: none;
  }

  .hamburger {
    display: block;
  }
}

Με αυτόν τον τρόπο η ιστοσελίδα γίνεται εύχρηστη και ευανάγνωστη σε κινητές συσκευές.

Responsive Εικόνες

Για να προσαρμόζονται οι εικόνες αυτόματα στο πλάτος της οθόνης, χρησιμοποιούμε: 

img {
  max-width: 100%;
  height: auto;
}

Έτσι, οι εικόνες δεν ξεφεύγουν από το πλάτος του κοντέινερ τους και φαίνονται σωστά παντού.

Flexbox & Grid για Responsive Layouts

Για πιο σύνθετες responsive διατάξεις, χρησιμοποιούμε Flexbox ή CSS Grid: 

.container {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}
.item {
  flex: 1 1 200px;
  min-width: 180px;
}

Με το flex-wrap: wrap; τα στοιχεία "σπάνε" σε νέα γραμμή όταν δεν χωράνε, διατηρώντας την ευελιξία της διάταξης σε κάθε οθόνη.

Συχνά Λάθη Αρχαρίων στο Responsive Design

  • Ξεχνούν το <meta name="viewport"> στο <head>.
  • Δεν χρησιμοποιούν media queries ή τα βάζουν λάθος.
  • Ορίζουν σταθερά πλάτη (π.χ. width: 600px;) αντί για ποσοστά ή max-width.
  • Δεν ελέγχουν τη σελίδα σε διαφορετικές συσκευές/αναλύσεις.

Άσκηση για Εξάσκηση

  1. Φτιάξε μια σελίδα με τρεις εικόνες στη σειρά.
  2. Χρησιμοποίησε Flexbox για να εμφανίζονται οριζόντια σε μεγάλες οθόνες και κάθετα σε κινητά (με media query).
  3. Βεβαιώσου ότι οι εικόνες είναι responsive (max-width: 100%;).

Αναλυτική σκέψη για κάθε έννοια

Έννοια Περιγραφή / Ρόλος Παράδειγμα
document Αντικείμενο που αναπαριστά όλο το HTML της σελίδας (DOM). document.getElementById("myText")
getElementById Μέθοδος που επιστρέφει το στοιχείο με το συγκεκριμένο id. document.getElementById("myText")
textContent Ιδιότητα που αλλάζει το κείμενο ενός στοιχείου. element.textContent = "Νέο κείμενο";
alert() Μέθοδος που εμφανίζει παράθυρο με μήνυμα στον χρήστη. alert("Μήνυμα");
addEventListener Μέθοδος που προσθέτει ακροατή συμβάντος σε ένα στοιχείο. button.addEventListener("click", function() {...})
function Ομάδα εντολών που εκτελούνται μαζί όταν καλείται η συνάρτηση ή συμβαίνει ένα event. function showAlert() {...}
let Δήλωση μεταβλητής (αποθηκεύει προσωρινά τιμές). let el = document.getElementById("colorText");
style Αντικείμενο που επιτρέπει να αλλάξουμε CSS ιδιότητες από τη JavaScript. el.style.color = "#d84315";
classList.add Μέθοδος που προσθέτει CSS κλάση σε στοιχείο. el.classList.add("highlight");
createElement Δημιουργεί νέο στοιχείο HTML μέσω JavaScript. document.createElement("p");
appendChild Προσθέτει ένα στοιχείο ως παιδί σε άλλο στοιχείο. container.appendChild(newP);
remove() Αφαιρεί το στοιχείο από το DOM. element.remove();
setTimeout Εκτελεί μια συνάρτηση μετά από καθυστέρηση. setTimeout(function(){...}, 500);

Σύνοψη

  • Με τη JavaScript μπορούμε να αλλάζουμε το περιεχόμενο, το στυλ και τη δομή της σελίδας (DOM).
  • Οι συναρτήσεις οργανώνουν τον κώδικα και εκτελούν ενέργειες όταν καλούνται ή όταν συμβαίνει κάποιο γεγονός (event).
  • Οι μέθοδοι και οι ιδιότητες του DOM μας επιτρέπουν να αλληλεπιδρούμε με τα στοιχεία της σελίδας δυναμικά.
  • Η χρήση διαφορετικών τρόπων (onclick, addEventListener) μας βοηθά να κατανοήσουμε τη δύναμη της JavaScript και του DOM.

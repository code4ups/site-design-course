JavaScript - Εισαγωγή στη Δυναμική Συμπεριφορά Ιστοσελίδων

Η JavaScript είναι η γλώσσα προγραμματισμού που δίνει ζωή και διαδραστικότητα στις ιστοσελίδες. Με τη JavaScript μπορείτε να δημιουργήσετε δυναμικό περιεχόμενο, να αντιδράτε σε ενέργειες του χρήστη και να επικοινωνείτε με εξυπηρετητές χωρίς να φορτώνετε ξανά τη σελίδα.

Τι είναι η JavaScript;

Η JavaScript είναι μια γλώσσα σεναρίων (scripting language) που εκτελείται στον φυλλομετρητή (browser) του χρήστη. Είναι βασικό εργαλείο για την ανάπτυξη σύγχρονων ιστοσελίδων και web εφαρμογών.

  • Τροποποιεί το HTML και CSS σε πραγματικό χρόνο.
  • Αντιδρά σε ενέργειες όπως κλικ, πληκτρολόγηση, μετακινήσεις ποντικιού.
  • Επιτρέπει επικοινωνία με διακομιστές (AJAX, Fetch API).
  • Υποστηρίζει προγραμματιστικές έννοιες όπως μεταβλητές, συναρτήσεις, βρόχους, αντικείμενα.

Απλό Παράδειγμα JavaScript

<!DOCTYPE html>
<html>
  <head>
    <title>Παράδειγμα JavaScript</title>
    <script>
      function sayHello() {
        alert("Γειά σου Κόσμε!");
      }
    </script>
  </head>
  <body>
    <button onclick="sayHello()">Κάνε Κλικ</button>
  </body>
</html>

Όταν ο χρήστης πατάει το κουμπί, καλείται η συνάρτηση sayHello() που εμφανίζει ένα παράθυρο με μήνυμα.

Βασικές Έννοιες της JavaScript

  • Μεταβλητές: Αποθηκεύουν δεδομένα. Π.χ. let name = "Γιάννης";
  • Συναρτήσεις: Μπλοκ κώδικα που εκτελούν συγκεκριμένη εργασία.
  • Εκδηλώσεις (Events): Ενέργειες που προκαλούν κώδικα, π.χ. κλικ, φόρτωση σελίδας.
  • DOM (Document Object Model): Αντιπροσωπεύει τη δομή της HTML και επιτρέπει την αλλαγή της μέσω JavaScript.

Πώς να προσθέσετε JavaScript στη σελίδα σας

Υπάρχουν τρεις βασικοί τρόποι:

  1. Ενσωματωμένος κώδικας μέσα σε <script> στο HTML.
  2. Εξωτερικό αρχείο JavaScript που συνδέεται με <script src="filename.js"></script>.
  3. Inline κώδικας μέσα σε HTML στοιχεία (π.χ. onclick="function()").

Συχνά Λάθη Αρχαρίων στη JavaScript

  • Ξεχνούν το ; στο τέλος των εντολών.
  • Μπερδεύουν var, let και const.
  • Προσπαθούν να χρησιμοποιήσουν στοιχεία HTML πριν φορτώσει η σελίδα (πρέπει να περιμένουν το DOM).
  • Δεν ελέγχουν για λάθη και δεν χρησιμοποιούν εργαλεία debugging.

Άσκηση για Εξάσκηση

Δες ΕΔΩ
  1. Δημιουργήστε ένα αρχείο script.js και γράψτε μια συνάρτηση που αλλάζει το κείμενο ενός στοιχείου HTML όταν πατιέται ένα κουμπί.
  2. Προσθέστε ένα κουμπί στη σελίδα σας και συνδέστε το με τη συνάρτηση.
  3. Δοκιμάστε να εμφανίσετε ένα alert με μήνυμα όταν ο χρήστης κάνει κλικ.

Απαντήσεις στις Ασκήσεις

  1. Συνάρτηση που αλλάζει το κείμενο ενός στοιχείου HTML όταν πατιέται κουμπί
    HTML (π.χ. στο body): 
    <p id="myText">Αυτό είναι το αρχικό κείμενο.</p>
<button onclick="changeText()">Άλλαξε το κείμενο</button>
    JavaScript (στο αρχείο script.js): 
    // Συνάρτηση που αλλάζει το κείμενο του στοιχείου με id "myText"
function changeText() {
    // Βρίσκουμε το στοιχείο με id "myText" και αλλάζουμε το κείμενό του
    document.getElementById("myText").textContent = "Το κείμενο άλλαξε με JavaScript!";
}
  2. Προσθήκη κουμπιού και σύνδεσή του με τη συνάρτηση
    Βλέπε παραπάνω παράδειγμα: το κουμπί έχει onclick="changeText()" που καλεί τη συνάρτηση όταν πατηθεί.
  3. Εμφάνιση alert με μήνυμα όταν ο χρήστης κάνει κλικ
    Μπορείς να προσθέσεις ένα δεύτερο κουμπί ή να το ενσωματώσεις στην ίδια συνάρτηση: 
    <button onclick="showAlert()">Δείξε Μήνυμα</button>
            
    // Συνάρτηση που εμφανίζει alert με μήνυμα
function showAlert() {
    alert("Καλώς ήρθες στη σελίδα μας!");
}

Παράδειγμα ολοκληρωμένου HTML με τα παραπάνω: 

<!-- Μέσα στο <body> της σελίδας -->
<p id="myText">Αυτό είναι το αρχικό κείμενο.</p>
<button onclick="changeText()">Άλλαξε το κείμενο</button>
<button onclick="showAlert()">Δείξε Μήνυμα</button>

<!-- Στο αρχείο script.js -->
function changeText() {
    document.getElementById("myText").textContent = "Το κείμενο άλλαξε με JavaScript!";
}

function showAlert() {
    alert("Καλώς ήρθες στη σελίδα μας!");
}

Σχόλιο:
Το πρώτο κουμπί αλλάζει το κείμενο της παραγράφου, το δεύτερο εμφανίζει μήνυμα. Μπορείς να χρησιμοποιήσεις και τις δύο συναρτήσεις ή να τις συνδυάσεις όπως θέλεις!

PHP & MySQL - Διασύνδεση και Βασικές Εντολές

Σε αυτή την ενότητα θα μάθετε πώς να χρησιμοποιείτε τη PHP για να δημιουργείτε δυναμικές ιστοσελίδες που επικοινωνούν με βάσεις δεδομένων MySQL. Η PHP είναι γλώσσα προγραμματισμού που εκτελείται στον διακομιστή (server-side), ενώ η MySQL είναι ένα από τα πιο διαδεδομένα συστήματα διαχείρισης βάσεων δεδομένων.

Τι είναι η PHP και η MySQL;

  • PHP: Γλώσσα προγραμματισμού για τον server, ιδανική για δυναμικές ιστοσελίδες, φόρμες, διαχείριση χρηστών κ.ά.
  • MySQL: Σύστημα διαχείρισης βάσεων δεδομένων όπου αποθηκεύονται οργανωμένα τα δεδομένα της ιστοσελίδας (π.χ. χρήστες, άρθρα, σχόλια).

Σύνδεση σε Βάση Δεδομένων με PHP

<?php
$host = "localhost";
$user = "root";
$password = "";
$dbname = "my_database";

$conn = new mysqli($host, $user, $password, $dbname);

// Έλεγχος σύνδεσης
if ($conn->connect_error) {
    die("Σφάλμα σύνδεσης: " . $conn->connect_error);
}
echo "Η σύνδεση ήταν επιτυχής!";
?>
  • $host: Ο server της βάσης (συνήθως localhost σε τοπικό περιβάλλον).
  • $user: Όνομα χρήστη της βάσης (π.χ. root).
  • $password: Κωδικός χρήστη (συχνά κενός τοπικά).
  • $dbname: Όνομα της βάσης που έχεις δημιουργήσει.

Ο παραπάνω κώδικας χρησιμοποιεί το αντικείμενο mysqli για να συνδεθεί στη βάση. Αν η σύνδεση αποτύχει, εμφανίζεται μήνυμα λάθους.

Εκτέλεση Ερωτήματος SQL (SELECT)

<?php
$sql = "SELECT id, name FROM users";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
    while($row = $result->fetch_assoc()) {
        echo "ID: " . $row["id"] . " - Όνομα: " . $row["name"] . "<br>";
    }
} else {
    echo "0 αποτελέσματα";
}
$conn->close();
?>

Με το $conn->query() εκτελείται το ερώτημα SQL. Με το fetch_assoc() διαβάζεις κάθε γραμμή αποτελεσμάτων ως πίνακα.

Εισαγωγή Δεδομένων (INSERT)

<?php
$sql = "INSERT INTO users (name) VALUES ('Γιάννης')";
if ($conn->query($sql) === TRUE) {
    echo "Η εγγραφή προστέθηκε με επιτυχία!";
} else {
    echo "Σφάλμα: " . $conn->error;
}
?>

Με το INSERT INTO προσθέτεις νέα δεδομένα στον πίνακα users.

Ασφάλεια: Προετοιμασμένες Δηλώσεις

Για να αποφύγεις επιθέσεις SQL injection, χρησιμοποίησε prepared statements: 

<?php
$stmt = $conn->prepare("INSERT INTO users (name) VALUES (?)");
$stmt->bind_param("s", $name);
$name = "Μαρία";
$stmt->execute();
$stmt->close();
?>

Συχνά Λάθη Αρχαρίων σε PHP & MySQL

  • Ξεχνούν να κλείσουν τη σύνδεση με $conn->close();
  • Δεν ελέγχουν αν η σύνδεση ή το ερώτημα πέτυχε.
  • Γράφουν SQL εντολές απευθείας χωρίς ασφάλεια.
  • Μπερδεύουν τα ονόματα των πεδίων ή των πινάκων.

Άσκηση για Εξάσκηση

Δες ΕΔΩ
  1. Δημιούργησε μια βάση δεδομένων my_database και έναν πίνακα users με πεδία id (AUTO_INCREMENT) και name (VARCHAR).
  2. Γράψε κώδικα PHP για να προσθέσεις έναν νέο χρήστη και να εμφανίσεις όλους τους χρήστες.
  3. Δοκίμασε να κάνεις το ίδιο με προετοιμασμένες δηλώσεις.
  4. Εκτέλεσε όλα τα παραπάνω βήματα χρησιμοποιώντας μόνο SQL εντολές μέσα στο phpMyAdmin (χωρίς PHP).

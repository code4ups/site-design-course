CSS3 & Flexbox - Μορφοποίηση και Διάταξη Ιστοσελίδων

Σε αυτή την ενότητα θα μάθετε τα βασικά της CSS3 για τη μορφοποίηση ιστοσελίδων και θα γνωρίσετε το Flexbox για ευέλικτη και μοντέρνα διάταξη περιεχομένου.

Τι είναι η CSS;

Η CSS (Cascading Style Sheets) είναι η γλώσσα που χρησιμοποιούμε για να δίνουμε μορφή και στυλ στις ιστοσελίδες μας: χρώματα, γραμματοσειρές, αποστάσεις, σκιές, στρογγυλεμένες γωνίες και πολλά άλλα.

  • Διαχωρίζει το περιεχόμενο (HTML) από την εμφάνιση (CSS).
  • Κάνει τις ιστοσελίδες πιο όμορφες και ευανάγνωστες.
  • Επιτρέπει την προσαρμογή της εμφάνισης σε κινητά, tablet και υπολογιστές.

Παράδειγμα Βασικού CSS

<style>
body {
  background-color: #f0f8ff;
  color: #222;
  font-family: Arial, sans-serif;
}
h1 {
  color: #0077cc;
}
.button {
  background: #ff9800;
  color: #fff;
  padding: 10px 18px;
  border: none;
  border-radius: 6px;
  cursor: pointer;
}
.button:hover {
  background: #e65100;
}
</style>

Το CSS μπορεί να γραφτεί μέσα σε ένα αρχείο .css, σε ένα <style> στο <head> ή απευθείας σε κάθε στοιχείο (συνήθως αποφεύγεται).

Συχνές Ιδιότητες CSS

  • color: Χρώμα κειμένου
  • background-color: Χρώμα φόντου
  • font-size: Μέγεθος γραμματοσειράς
  • padding: Εσωτερικό περιθώριο
  • margin: Εξωτερικό περιθώριο
  • border: Περιγράμματα
  • border-radius: Στρογγυλεμένες γωνίες
  • box-shadow: Σκιά
  • text-align: Στοίχιση κειμένου

Συχνά Λάθη Αρχαρίων στη CSS

  • Ξεχνούν να βάλουν το σωστό selector (π.χ. .button αντί για button).
  • Χρησιμοποιούν λάθος συντακτικό (π.χ. ξεχνούν το ; στο τέλος κάθε γραμμής).
  • Δεν συνδέουν σωστά το CSS αρχείο με το HTML.
  • Γράφουν CSS απευθείας σε κάθε στοιχείο (inline style).

Άσκηση για Εξάσκηση

  1. Δημιούργησε ένα αρχείο style.css και σύνδεσέ το με το HTML σου.
  2. Δώσε χρώμα φόντου στη σελίδα και άλλαξε το χρώμα στους τίτλους.
  3. Φτιάξε ένα κουμπί με δικό σου στυλ και δοκίμασε να αλλάζει χρώμα όταν το ποντίκι περνάει από πάνω.

Flexbox - Ευέλικτη Διάταξη με CSS3

Το Flexbox είναι ένα μοντέρνο σύστημα διάταξης που επιτρέπει τη δημιουργία ευέλικτων και δυναμικών διατάξεων χωρίς floats ή positioning. Είναι ιδανικό για στοίχιση στοιχείων στη σειρά ή σε στήλες και για responsive σχεδίαση.

Παράδειγμα Flexbox

<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
</div>
        
.flex-container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 16px;
  background: #e3f2fd;
  padding: 20px;
}
.flex-item {
  background: #90caf9;
  padding: 20px 30px;
  border-radius: 8px;
  font-size: 1.2em;
}
  • display: flex – Δηλώνει το κοντέινερ ως flexbox.
  • justify-content – Στοίχιση στον κύριο άξονα (οριζόντια).
  • align-items – Στοίχιση στον εγκάρσιο άξονα (κάθετα).
  • gap – Απόσταση μεταξύ των flex items.

Συχνά Λάθη με Flexbox

  • Ξεχνούν να ορίσουν display: flex στον γονικό κοντέινερ.
  • Χρησιμοποιούν λάθος ιδιότητες (π.χ. align-items αντί για align-content).
  • Δεν χρησιμοποιούν το gap για αποστάσεις μεταξύ στοιχείων.

Άσκηση Flexbox

  1. Δημιούργησε ένα div με κλάση flex-container και τρία div με κλάση flex-item μέσα του.
  2. Εφάρμοσε το CSS για να τα στοιχίσεις οριζόντια και να έχουν αποστάσεις μεταξύ τους.
  3. Δοκίμασε να αλλάξεις το justify-content σε space-between ή flex-end και παρατήρησε τη διαφορά.

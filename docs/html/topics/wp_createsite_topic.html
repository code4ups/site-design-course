Δημιουργία Ιστοσελίδας με WordPress (Creating a Website with WordPress)

Εισαγωγή (Introduction)

Σε αυτό το μάθημα θα μάθεις πώς να δημιουργήσεις ένα σύγχρονο, λειτουργικό website χωρίς να χρειάζεσαι γνώσεις προγραμματισμού, χρησιμοποιώντας το WordPress. Ο οδηγός είναι κατάλληλος για αρχάριους και σε καθοδηγεί βήμα-βήμα, από την επιλογή domain και hosting, μέχρι την τελική διαμόρφωση και δημοσίευση της ιστοσελίδας σου.

Τι είναι το WordPress; (What is WordPress?)

Το WordPress είναι ένα σύστημα διαχείρισης περιεχομένου (CMS) που σου επιτρέπει να δημιουργήσεις και να διαχειριστείς ιστοσελίδες εύκολα, χωρίς να γράψεις ούτε μία γραμμή κώδικα. Είναι το πιο δημοφιλές CMS παγκοσμίως, χρησιμοποιείται από πάνω από το 40% των ιστοσελίδων και διατίθεται δωρεάν.

  • Δεν χρειάζεται να γνωρίζεις HTML, CSS ή JavaScript.
  • Όλη η διαχείριση γίνεται γραφικά, μέσα από το διαχειριστικό περιβάλλον.
  • Είναι ιδανικό για blogs, επαγγελματικές ιστοσελίδες, e-shop και πολλά άλλα.

Απόκτηση Domain και Hosting (Getting a Domain and Hosting)

Για να είναι το site σου προσβάσιμο στο διαδίκτυο, χρειάζεσαι δύο πράγματα:

  • Domain name: Η διεύθυνση που θα πληκτρολογούν οι επισκέπτες (π.χ. toonomasou.gr).
  • Hosting: Ο χώρος όπου θα φιλοξενείται το site και τα αρχεία του.

Βήματα:

  1. Επίλεξε έναν αξιόπιστο πάροχο hosting που υποστηρίζει WordPress (π.χ. Hostinger, Papaki, Top.Host, SiteGround).
  2. Κατά την αγορά hosting, μπορείς να κατοχυρώσεις και το domain σου.
  3. Αγόρασε το πακέτο που καλύπτει τις ανάγκες σου (συνήθως τα βασικά πακέτα είναι αρκετά για αρχή).
  4. Μετά την αγορά, θα λάβεις email με στοιχεία σύνδεσης στο control panel του hosting σου (συνήθως cPanel ή hPanel).

Συμβουλή: Το domain κοστίζει περίπου 10-20€ το χρόνο και το hosting περίπου 3-5€ το μήνα για ένα απλό site.

Εγκατάσταση WordPress (Installing WordPress)

Υπάρχουν δύο βασικοί τρόποι για να εγκαταστήσεις το WordPress στον server σου:

  1. Αυτόματη εγκατάσταση μέσω του hosting:
    • Συνδέσου στο control panel του hosting σου (π.χ. cPanel/hPanel).
    • Βρες το εργαλείο “WordPress Installer” ή “Softaculous”.
    • Επίλεξε το domain σου, όρισε όνομα site, όνομα χρήστη και κωδικό διαχειριστή.
    • Πάτησε “Εγκατάσταση” και περίμενε να ολοκληρωθεί.
  2. Χειροκίνητη εγκατάσταση:
    • Κατέβασε το WordPress από το επίσημο site.
    • Ανέβασε τα αρχεία στον server μέσω FTP.
    • Δημιούργησε βάση δεδομένων MySQL από το control panel.
    • Άνοιξε τον browser και πήγαινε στη διεύθυνση του site σου για να ξεκινήσεις τη διαδικασία εγκατάστασης.
    • Συμπλήρωσε τα στοιχεία της βάσης και ολοκλήρωσε την εγκατάσταση.

Σημείωση: Οι περισσότεροι αρχάριοι επιλέγουν την αυτόματη εγκατάσταση γιατί είναι πιο απλή και γρήγορη.

Ρύθμιση Προσαρμοσμένου Email (Setting Up Custom Email)

Με το domain σου μπορείς να δημιουργήσεις επαγγελματικές διευθύνσεις email (π.χ. info@toonomasou.gr).
Βήματα:

  1. Συνδέσου στο control panel του hosting σου.
  2. Βρες την ενότητα “Email Accounts” ή “Λογαριασμοί Email”.
  3. Δημιούργησε νέο email, όρισε κωδικό και αποθήκευσε.
  4. Μπορείς να ελέγχεις τα email σου μέσω webmail ή να τα προωθείς σε άλλη διεύθυνση (π.χ. Gmail).
  5. Για να στέλνεις/λαμβάνεις email, μπορείς να χρησιμοποιήσεις το webmail του hosting ή να ρυθμίσεις το email σε εφαρμογή (Outlook, Gmail κ.λπ.).

Συμβουλή: Επαλήθευσε το email σου αν ζητηθεί, ώστε να λειτουργεί σωστά η αποστολή και λήψη.

Εξερεύνηση Πίνακα Ελέγχου WordPress (Exploring WordPress Dashboard)

Αφού ολοκληρωθεί η εγκατάσταση, συνδέσου στο διαχειριστικό του WordPress (π.χ. yourdomain.gr/wp-admin).
Στον πίνακα ελέγχου θα βρεις:

  • Το αριστερό μενού με επιλογές για άρθρα, σελίδες, εμφάνιση, πρόσθετα, χρήστες, ρυθμίσεις κ.ά.
  • Την κεντρική σελίδα με γρήγορες επιλογές και ειδοποιήσεις.
  • Εδώ μπορείς να προσθέσεις περιεχόμενο, να αλλάξεις το θέμα, να εγκαταστήσεις πρόσθετα και να ρυθμίσεις το site σου.

Συμβουλή: Η μαύρη μπάρα στο πάνω μέρος της σελίδας εμφανίζεται μόνο όταν είσαι συνδεδεμένος ως διαχειριστής.

Ενημέρωση Ρυθμίσεων (Updating Settings)

Πριν ξεκινήσεις να προσθέτεις περιεχόμενο, καλό είναι να ρυθμίσεις βασικές επιλογές:

  1. Πήγαινε στο “Ρυθμίσεις” → “Γενικά”. Συμπλήρωσε τίτλο ιστοσελίδας, περιγραφή, email διαχειριστή, ζώνη ώρας και γλώσσα.
  2. Στο “Ρυθμίσεις” → “Σύνδεσμοι” (Permalinks), επέλεξε “Όνομα άρθρου” για φιλικές διευθύνσεις URL (SEO friendly).
  3. Ρύθμισε αν θέλεις να επιτρέπονται σχόλια, ποια θα είναι η αρχική σελίδα κ.λπ.
  4. Αποθήκευσε τις αλλαγές.

Εγκατάσταση Πρόσθετων (Installing Plugins)

Τα πρόσθετα (plugins) επεκτείνουν τις δυνατότητες του WordPress. Για παράδειγμα, μπορείς να προσθέσεις φόρμα επικοινωνίας, SEO, ασφάλεια, gallery φωτογραφιών κ.ά.

  1. Πήγαινε στο “Πρόσθετα” → “Νέο Πρόσθετο”.
  2. Αναζήτησε το πρόσθετο που θέλεις (π.χ. “Elementor”, “WPForms”).
  3. Πάτησε “Εγκατάσταση” και μετά “Ενεργοποίηση”.
  4. Διαχειρίσου τα πρόσθετα από το ίδιο μενού.
  5. Αφαίρεσε όσα πρόσθετα δεν χρειάζεσαι για να κρατήσεις το site ελαφρύ και ασφαλές.

Συμβουλή: Εγκατέστησε μόνο όσα πρόσθετα χρειάζεσαι πραγματικά.

Εγκατάσταση Θέματος (Installing Theme)

Το θέμα (theme) καθορίζει την εμφάνιση του site σου. Μπορείς να διαλέξεις ανάμεσα σε χιλιάδες δωρεάν ή επί πληρωμή θέματα.

  1. Πήγαινε στο “Εμφάνιση” → “Θέματα”.
  2. Πάτησε “Προσθήκη νέου” για να δεις τα διαθέσιμα θέματα.
  3. Δοκίμασε προεπισκόπηση (Preview) και εγκατέστησε το θέμα που σου αρέσει.
  4. Μετά την εγκατάσταση, πάτησε “Ενεργοποίηση”.
  5. Για περισσότερες δυνατότητες, προτείνεται το θέμα Astra που συνεργάζεται άψογα με το Elementor.

Δημιουργία Λογοτύπου (Creating a Logo)

Ένα καλό λογότυπο δίνει ταυτότητα στην ιστοσελίδα σου.

  1. Μπες σε ένα εργαλείο δημιουργίας λογοτύπου, όπως το Canva ή το Looka.
  2. Επίλεξε διαστάσεις (συνήθως τετράγωνο, π.χ. 500x500px).
  3. Σχεδίασε το λογότυπο με κείμενο, εικόνες ή εικονίδια που ταιριάζουν στο θέμα του site σου.
  4. Κατέβασε το λογότυπο σε μορφή PNG με διαφάνεια (αν είναι δυνατόν).

Προσθήκη Λογοτύπου στο Site (Adding Logo to Website)

Για να προσθέσεις το λογότυπο σου:

  1. Πήγαινε στο “Εμφάνιση” → “Προσαρμογή”.
  2. Επίλεξε “Ταυτότητα Ιστοσελίδας”.
  3. Ανέβασε το αρχείο του λογοτύπου και κάνε αποθήκευση.
  4. Το λογότυπο θα εμφανίζεται πλέον στην κεφαλίδα (header) του site σου.

Επεξεργασία Site με Elementor (Editing Site with Elementor)

Το Elementor είναι ένα πρόσθετο που σου επιτρέπει να διαμορφώνεις σελίδες με drag & drop, χωρίς να γράφεις κώδικα.

  1. Εγκατέστησε και ενεργοποίησε το Elementor.
  2. Πήγαινε σε μια σελίδα ή άρθρο και πάτησε “Επεξεργασία με Elementor”.
  3. Χρησιμοποίησε τα blocks για να προσθέσεις κείμενα, εικόνες, κουμπιά κ.λπ.
  4. Μπορείς να αλλάξεις χρώματα, γραμματοσειρές και διάταξη εύκολα.
  5. Αποθήκευσε τις αλλαγές και δες άμεσα το αποτέλεσμα.

Προσθήκη Προσαρμοσμένου HTML (Adding Custom HTML to Site)

Αν θέλεις να προσθέσεις δικό σου κώδικα HTML (π.χ. widgets, scripts):

  1. Στον Gutenberg editor, πρόσθεσε block “Προσαρμοσμένο HTML”.
  2. Επικόλλησε τον κώδικα HTML που θέλεις.
  3. Με το Elementor, χρησιμοποίησε το widget “HTML” και πρόσθεσε τον κώδικα.
  4. Πάντα να ελέγχεις ότι ο κώδικας δεν προκαλεί προβλήματα στην εμφάνιση ή τη λειτουργία του site.

Ενημέρωση Μενού (Updating Menus)

Τα μενού βοηθούν τους επισκέπτες να περιηγούνται εύκολα στο site σου.

  1. Πήγαινε στο “Εμφάνιση” → “Μενού”.
  2. Δημιούργησε νέο μενού ή επεξεργάσου το υπάρχον.
  3. Πρόσθεσε σελίδες, άρθρα ή custom links στο μενού.
  4. Όρισε το μενού ως “Κύριο Μενού” (Primary Menu) για να εμφανίζεται στην πλοήγηση.
  5. Αποθήκευσε τις αλλαγές.

Προσθήκη Νέας Σελίδας (Adding a New Page)

Για να προσθέσεις νέα σελίδα:

  1. Πήγαινε στο “Σελίδες” → “Προσθήκη Νέας”.
  2. Γράψε τον τίτλο και το περιεχόμενο της σελίδας.
  3. Χρησιμοποίησε blocks για να προσθέσεις εικόνες, λίστες, κουμπιά κ.λπ.
  4. Πάτησε “Δημοσίευση” για να εμφανιστεί η σελίδα στο site.

Προσαρμογή Ονόματος Σελίδας (Customizing Page Name)

Μπορείς να αλλάξεις τον τίτλο και το URL (σύνδεσμο) κάθε σελίδας:

  1. Πριν δημοσιεύσεις, επεξεργάσου το πεδίο “Μόνιμος σύνδεσμος” (Permalink) κάτω από τον τίτλο.
  2. Δώσε ένα σύντομο και περιγραφικό URL (π.χ. epikoinonia αντί για page123).
  3. Αποθήκευσε τις αλλαγές.

Επεξεργασία Σελίδας "Σχετικά" (Editing About Page)

Η σελίδα “Σχετικά” παρουσιάζει εσένα ή την επιχείρησή σου στους επισκέπτες:

  1. Πήγαινε στη σελίδα “Σχετικά” και πάτησε “Επεξεργασία”.
  2. Γράψε λίγα λόγια για εσένα, την ομάδα ή την αποστολή του site.
  3. Πρόσθεσε φωτογραφίες, βιογραφικό ή ιστορικό.
  4. Χρησιμοποίησε blocks για να κάνεις το περιεχόμενο πιο ελκυστικό.
  5. Αποθήκευσε τις αλλαγές.

Επεξεργασία Σελίδας "Επικοινωνία" (Editing Contact Page)

Η σελίδα “Επικοινωνία” επιτρέπει στους επισκέπτες να έρθουν σε επαφή μαζί σου:

  1. Δημιούργησε ή επεξεργάσου τη σελίδα “Επικοινωνία”.
  2. Πρόσθεσε φόρμα επικοινωνίας με plugin (π.χ. WPForms) ή απλά τα στοιχεία επικοινωνίας σου (email, τηλέφωνο).
  3. Δώσε σαφείς οδηγίες για το πώς μπορεί κάποιος να επικοινωνήσει μαζί σου.
  4. Αποθήκευσε τις αλλαγές.

Προσαρμογή Φόρμας Επικοινωνίας (Customizing Contact Form)

Για να προσαρμόσεις τη φόρμα επικοινωνίας:

  1. Άνοιξε το plugin της φόρμας (π.χ. WPForms).
  2. Πρόσθεσε ή αφαίρεσε πεδία (όνομα, email, μήνυμα κ.λπ.).
  3. Όρισε που θα στέλνονται τα μηνύματα (email διαχειριστή).
  4. Διαμόρφωσε το μήνυμα επιβεβαίωσης που βλέπει ο επισκέπτης μετά την αποστολή.
  5. Ενσωμάτωσε τη φόρμα στη σελίδα “Επικοινωνία” με shortcode ή block.

Ενημέρωση Υποσέλιδου (Updating Page Footer)

Το υποσέλιδο (footer) εμφανίζεται στο κάτω μέρος κάθε σελίδας:

  1. Πήγαινε στο “Εμφάνιση” → “Προσαρμογή” → “Υποσέλιδο” ή “Widgets”.
  2. Πρόσθεσε στοιχεία όπως πνευματικά δικαιώματα, social media, επιπλέον συνδέσμους ή πληροφορίες.
  3. Αποθήκευσε τις αλλαγές.

Κάνε το Site Responsive (Making Site Responsive)

Η responsive σχεδίαση εξασφαλίζει ότι το site σου φαίνεται σωστά σε κινητά, tablet και υπολογιστές.

  1. Επίλεξε θέμα που είναι responsive (τα περισσότερα σύγχρονα θέματα το υποστηρίζουν).
  2. Με το Elementor μπορείς να δεις και να προσαρμόσεις την εμφάνιση για κινητά/tablet.
  3. Δοκίμασε το site σου σε διάφορες συσκευές και βελτίωσε όπου χρειάζεται.

Προσθήκη Blog (Adding a Blog)

Το blog είναι μια ενότητα όπου μπορείς να δημοσιεύεις νέα, άρθρα ή ενημερώσεις.

  1. Πήγαινε στο “Άρθρα” → “Προσθήκη Νέου”.
  2. Γράψε το άρθρο σου και δημοσίευσέ το.
  3. Οργάνωσε τα άρθρα σε κατηγορίες (π.χ. Νέα, Ανακοινώσεις).
  4. Όρισε μια σελίδα ως “Blog” από τις ρυθμίσεις “Ανάγνωση”.

Δημιουργία Αναρτήσεων με AI Content Creator (Creating Posts with AI Content Creator)

Μπορείς να χρησιμοποιήσεις εργαλεία Τεχνητής Νοημοσύνης (AI) για να δημιουργήσεις περιεχόμενο πιο γρήγορα.

  1. Δοκίμασε εργαλεία όπως ChatGPT, Jasper ή το ενσωματωμένο AI του hosting σου (αν υπάρχει).
  2. Ζήτησε ιδέες για άρθρα, προσχέδια ή ακόμα και έτοιμα κείμενα.
  3. Πάντα διάβαζε και προσαρμόζε το περιεχόμενο πριν το δημοσιεύσεις.

Προσαρμογή Blog (Customizing Blog)

Μπορείς να διαμορφώσεις την εμφάνιση και τη λειτουργία του blog σου:

  1. Πήγαινε στο “Εμφάνιση” → “Προσαρμογή” → “Blog” ή “Αρχεία”.
  2. Άλλαξε τη διάταξη, τα χρώματα, τις γραμματοσειρές και τα widgets του blog.
  3. Πρόσθεσε widgets για πρόσφατα άρθρα, κατηγορίες, αναζήτηση κ.λπ.
  4. Δοκίμασε διάφορες ρυθμίσεις μέχρι να βρεις αυτό που σου ταιριάζει.

Συμπέρασμα (Conclusion)

Ακολουθώντας τα παραπάνω βήματα, μπορείς να δημιουργήσεις ένα σύγχρονο, λειτουργικό και επαγγελματικό website με WordPress, ακόμα κι αν είναι η πρώτη σου φορά. Μην φοβηθείς να πειραματιστείς και να εξερευνήσεις τις δυνατότητες του WordPress – οι επιλογές είναι σχεδόν απεριόριστες!

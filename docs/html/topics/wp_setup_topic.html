Εγκατάσταση Wordpress σε Τοπικό Server με XAMPP (Windows)

Δες το Γλωσσάρι

Τι θα μάθεις με αυτόν τον οδηγό

  • Τι είναι το XAMPP και γιατί το χρησιμοποιούμε για τοπικό server.
  • Πώς να εγκαταστήσεις και να ρυθμίσεις το XAMPP στα Windows.
  • Πώς να κατεβάσεις και να εγκαταστήσεις το Wordpress τοπικά.
  • Πώς να δημιουργήσεις βάση δεδομένων για το Wordpress μέσω phpMyAdmin.
  • Πώς να ολοκληρώσεις τη βασική εγκατάσταση και να μπεις στο διαχειριστικό περιβάλλον του Wordpress.

Αναλυτικός Οδηγός Εγκατάστασης Wordpress (βήμα-βήμα)

  1. Κατέβασε και εγκατέστησε το XAMPP
    • Μεταβείτε στη σελίδα https://www.apachefriends.org/download.html και κατέβασε την έκδοση για Windows.
    • Τρέξε το αρχείο εγκατάστασης και ακολούθησε τα βήματα (προτείνεται να το εγκαταστήσεις στον φάκελο C:\xampp).
    • Μετά την εγκατάσταση, άνοιξε το XAMPP Control Panel και ξεκίνα τα modules Apache και MySQL πατώντας τα αντίστοιχα κουμπιά "Start".
  2. Κατέβασε το Wordpress
    • Πήγαινε στη σελίδα https://el.wordpress.org/download/ και κατέβασε το αρχείο ZIP της τελευταίας έκδοσης.
    • Αποσυμπίεσε το αρχείο ZIP.
    • Μετονόμασε τον φάκελο (π.χ. σε mywp) για να είναι το όνομα του site σου.
    • Μετακίνησε τον φάκελο mywp μέσα στον φάκελο C:\xampp\htdocs\.
  3. Δημιούργησε νέα βάση δεδομένων στο phpMyAdmin
    • Άνοιξε τον browser και πήγαινε στη διεύθυνση http://localhost/phpmyadmin.
    • Πάτησε στο "Νέα" (New) στην αριστερή στήλη.
    • Στο πεδίο "Όνομα βάσης δεδομένων" γράψε π.χ. mywp_db και πάτησε "Δημιουργία" (Create).
    • Δεν χρειάζεται να δημιουργήσεις πίνακες – το Wordpress θα το κάνει αυτόματα.
  4. Εκκίνηση της εγκατάστασης Wordpress
    • Άνοιξε τον browser και πήγαινε στη διεύθυνση http://localhost/mywp (ή το όνομα φακέλου που έδωσες).
    • Θα εμφανιστεί η αρχική οθόνη εγκατάστασης του Wordpress. Επίλεξε γλώσσα και πάτησε "Συνέχεια".
    • Συμπλήρωσε τα στοιχεία της βάσης δεδομένων:
      • Όνομα βάσης δεδομένων: mywp_db
      • Όνομα χρήστη: root
      • Κωδικός: (άφησέ το κενό)
      • Διακομιστής βάσης: localhost
      • Πρόθεμα πινάκων: wp_ (άφησέ το ως έχει)
    • Πάτησε "Υποβολή" και μετά "Εκκίνηση εγκατάστασης".
    • Στην επόμενη οθόνη, συμπλήρωσε τα στοιχεία του site (τίτλος, όνομα χρήστη διαχειριστή, κωδικός, email).
    • Ολοκλήρωσε την εγκατάσταση και συνδέσου στο διαχειριστικό με τα στοιχεία που έβαλες.
  5. Σύνδεση στο διαχειριστικό Wordpress
    • Για να μπεις στο διαχειριστικό, πήγαινε στη διεύθυνση http://localhost/mywp/wp-admin.
    • Συνδέσου με το όνομα χρήστη και τον κωδικό που όρισες.
    • Τώρα μπορείς να αρχίσεις να δημιουργείς και να επεξεργάζεσαι τοπικά το site σου!

Συχνές Ερωτήσεις και Συμβουλές

  • Δεν ξεκινάει το Apache ή το MySQL στο XAMPP: Βεβαιώσου ότι δεν τρέχουν άλλα προγράμματα που χρησιμοποιούν τις ίδιες θύρες (π.χ. Skype, IIS). Κλείσε τα ή άλλαξε τις θύρες στο XAMPP.
  • Ξέχασα τον κωδικό διαχειριστή Wordpress: Μπορείς να τον επαναφέρεις από τη βάση δεδομένων μέσω phpMyAdmin ή να δημιουργήσεις νέο χρήστη με SQL ερώτημα.
  • Το Wordpress ζητάει ξανά τα στοιχεία της βάσης: Έλεγξε ότι τα στοιχεία στη φόρμα είναι σωστά και ότι η βάση υπάρχει.
  • Γιατί να δουλέψω τοπικά με XAMPP; Είναι ασφαλές περιβάλλον για δοκιμές χωρίς να επηρεάζεις το πραγματικό site σου στο internet.

Βασικές Έννοιες που θα συναντήσεις

Έννοια Περιγραφή / Ρόλος Παράδειγμα / Σημείωση
XAMPP Τοπικός server που περιλαμβάνει Apache (web server), MySQL/MariaDB (βάση), PHP, phpMyAdmin. Εγκαθίσταται σε Windows, Linux ή Mac.
htdocs Κύριος φάκελος όπου βάζουμε τα αρχεία των sites μας στο XAMPP. C:\xampp\htdocs\mywp
phpMyAdmin Εργαλείο για διαχείριση βάσεων δεδομένων μέσω browser. http://localhost/phpmyadmin
Βάση δεδομένων Αποθηκεύει όλα τα δεδομένα του Wordpress (άρθρα, χρήστες, ρυθμίσεις). mywp_db
wp-admin Διαχειριστικό περιβάλλον Wordpress. http://localhost/mywp/wp-admin
wp-config.php Αρχείο ρυθμίσεων της εγκατάστασης του Wordpress. Δημιουργείται αυτόματα κατά την εγκατάσταση.

Σύνοψη

  • Η εγκατάσταση Wordpress σε τοπικό server με XAMPP είναι ιδανική για δοκιμές και εκμάθηση.
  • Ακολουθώντας τα βήματα του οδηγού, μπορείς να έχεις ένα πλήρες site Wordpress στον υπολογιστή σου χωρίς να χρειαστείς hosting ή domain.
  • Όλες οι αλλαγές που κάνεις είναι τοπικές και δεν επηρεάζουν το διαδίκτυο.
  • Όταν είσαι έτοιμος, μπορείς να μεταφέρεις το site σου σε πραγματικό server.

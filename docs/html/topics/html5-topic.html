HTML5 - Δημιουργία Δομής Ιστοσελίδας

Σε αυτή την ενότητα θα μάθετε πώς να δημιουργείτε τη βασική δομή μιας ιστοσελίδας χρησιμοποιώντας τη HTML5 (HyperText Markup Language 5). Η HTML είναι η "γλώσσα" με την οποία γράφουμε το σκελετό κάθε ιστοσελίδας στο διαδίκτυο.

Τι είναι η HTML5;

Η HTML5 είναι η πιο πρόσφατη έκδοση της HTML και προσφέρει νέες δυνατότητες για τη δημιουργία μοντέρνων, δυναμικών και προσβάσιμων ιστοσελίδων. Με την HTML5 μπορούμε να χρησιμοποιήσουμε νέες ετικέτες (tags), όπως <header>, <nav>, <main>, <footer>, που κάνουν τον κώδικα πιο κατανοητό και οργανωμένο.

Βασική Δομή Αρχείου HTML5

<!DOCTYPE html>
<html lang="el">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ο Τίτλος της Σελίδας</title>
  </head>
  <body>
    <h1>Καλώς ήρθατε στην ιστοσελίδα μου!</h1>
    <p>Αυτή είναι μια παραδειγματική παράγραφος.</p>
  </body>
</html>

Επεξήγηση:

  • <!DOCTYPE html>: Δηλώνει ότι χρησιμοποιούμε HTML5.
  • <html lang="el">: Όλη η σελίδα. Το lang="el" δηλώνει ότι η γλώσσα είναι τα Ελληνικά.
  • <head>: Περιέχει πληροφορίες για τη σελίδα (τίτλος, χαρακτήρες, responsive σχεδίαση).
  • <body>: Το ορατό περιεχόμενο της σελίδας.

Σημαντικές Ετικέτες HTML5

  • <header>: Η κεφαλίδα της σελίδας ή μιας ενότητας.
  • <nav>: Το μενού πλοήγησης.
  • <main>: Το κύριο περιεχόμενο της σελίδας.
  • <section>: Χωρίζει το περιεχόμενο σε ενότητες.
  • <article>: Αυτόνομο κομμάτι περιεχομένου (π.χ. άρθρο blog).
  • <aside>: Πληροφορίες στο πλάι (π.χ. sidebar).
  • <footer>: Το υποσέλιδο της σελίδας.

Αυτές οι ετικέτες βοηθούν στη δομή και την οργάνωση του κώδικα, αλλά και στη βελτίωση της προσβασιμότητας και του SEO.

Παράδειγμα: Μια Οργανωμένη Ιστοσελίδα με HTML5

<!DOCTYPE html>
<html lang="el">
  <head>
    <meta charset="UTF-8">
    <title>Παράδειγμα HTML5</title>
  </head>
  <body>
    <header>
      <h1>Η Σελίδα μου</h1>
    </header>
    <nav>
      <ul>
        <li><a href="#">Αρχική</a></li>
        <li><a href="#">Σχετικά</a></li>
      </ul>
    </nav>
    <main>
      <section>
        <h2>Καλώς ήρθατε!</h2>
        <p>Αυτή είναι μια οργανωμένη ιστοσελίδα με HTML5.</p>
      </section>
    </main>
    <footer>
      <p>© 2025 Εκπαιδευτικός Ιστότοπος</p>
    </footer>
  </body>
</html>

Συχνά Λάθη Αρχαρίων

  • Ξεχνούν το <!DOCTYPE html> στην αρχή.
  • Δεν κλείνουν σωστά τις ετικέτες (tags).
  • Χρησιμοποιούν λάθος σειρά στις ενότητες (<head>, <body>).
  • Δεν βάζουν lang="el" στο <html> για ελληνικές σελίδες.

Άσκηση για Εξάσκηση

  1. Δημιούργησε ένα νέο αρχείο index.html στον υπολογιστή σου.
  2. Γράψε τη βασική δομή HTML5 που έμαθες παραπάνω.
  3. Βάλε έναν τίτλο και μια παράγραφο με το όνομά σου.
  4. Άνοιξε το αρχείο στον browser σου και δες το αποτέλεσμα!
